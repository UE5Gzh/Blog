# 个人技术博客系统 (Personal Tech Blog System)

## What This Is

A personal technical blog system for sharing programming tutorials, code insights, and tech learnings. Built for developers who want to document and share their technical journey.

## Core Value

A clean, developer-focused blog platform that makes it easy to write, organize, and share technical content with good code presentation.

## Target Users

- Primary: Self (personal documentation and learning tracker)
- Secondary: Other developers looking for programming tutorials and code insights

## What Users Can Do

- Write and publish technical blog posts with markdown support
- Include code snippets with syntax highlighting (multiple languages)
- Organize content by categories and tags
- Read posts with clean, distraction-free typography
- Search through content

## v1 Scope (Hypotheses to Validate)

- [ ] User can create and publish blog posts with markdown
- [ ] User can write posts with embedded code snippets and syntax highlighting
- [ ] User can organize posts by categories and tags
- [ ] User can view posts in a clean, readable format
- [ ] User can search through published posts

## Out of Scope

- Multi-user / authentication system
- Comments and social features
- Analytics and tracking
- Newsletter subscription
- Paid content / monetization
